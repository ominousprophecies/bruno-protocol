Protocol Specification

.aet Format

Each function emits a fn_spec header block followed by the operational body. Security and energy attributes are encoded in the module header and per-function attribute lists.

.bru Recovery

The decompiler reconstructs module declarations, attribute brackets [entry: High], function signatures, map declarations, and scope epoch blocks.

GENXR_V1.4

Compiled manifests carry the GENXR_V1.4 / STRICT_MODE provenance header. The decompiler validates this marker before processing to ensure manifest integrity.

Protocol Specification

.bru Source

Input format for the Aether compiler. Extends the open-source Bruno API .bru format with SEML energy, security, and epoch attributes.

.aet Manifest

Proprietary compiled output. Each function emits a fn_spec header block followed by a verified operational body. Security and energy contracts are proven at compile time.

GENXR_V1.5

Verified baseline June 15, 2026. All 9 fixtures pass. Enforces InvalidSecurityEnergyCombination, SecurityTaintLeak, Krapivin lint, and strict external function contracts.
